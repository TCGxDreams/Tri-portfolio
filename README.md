# Tri-portfolio — The Tri Gazette

A beautiful, premium print-editorial styled communications portfolio using GSAP, Lenis Smooth Scroll, WebGL/Three.js fluid background simulations, dynamic canvas project hover waves, and a client-side Editor's Desk admin panel.

## Features
- **Print Editorial Design**: Minimalist newspaper grid layout, typography sizing protections, and responsive column blocks.
- **Sleek Scroll Animations**: Soft and full-pin chapter handoffs, ticker text velocity warp controls, and slide crossfades.
- **Editor's Desk (Admin Customizer)**: Direct color pickers and slot copywriters configured in `localStorage` to customize text and palettes.
- **Motion-Safe and Responsive**: Fully responsive grid stacks and prefers-reduced-motion fallback paths.

## Tech Stack
- HTML5 / CSS3 / Vanilla Javascript
- GSAP / ScrollTrigger
- Lenis Smooth Scroll
- Three.js (WebGL Fluid Simulation)
